```
1. 领取连续 1～3 个 Step
2. Codex / Cursor 实现
3. 跑对应测试、启动验证或页面手测
4. Codex 做代码审核
5. 必要节点交给 Claude Code 复审
6. 按审核意见修复
7. 重新跑测试(3-7直到没问题)
8. commit
9. 进入下一批 Step
```
